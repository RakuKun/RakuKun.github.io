Gunpla Wars - The Interactive
Gunpla Wars is a customized Gunpla model with remote controlling application built on mobile phone. This is a group project, I took the part as software designer and my partner Yiming Chen worked as hardware designer.
Goals
I and my partner had a feeling for Gunpla models and wanted it to be controlled to move like in its animation series. So we bought a Gunpla model and developed hardware and software based on it. We basically need to develop a system to send the remote control order and to catch and execute it on the Gunpla model.
Coding
I used Unity and Arduino to do software work. Unity application on mobile phone will get the gravity acceleration of the phone and send it via phone's bluetooth. Then a bluetooth receiver module on Gunpla model get the signal and transfer it to Arduino hardware-based orders.
