Project Google Stadia (ongoing)
In March 2019 Google Stadia announced a platform exclusive feature called State Share. This technology leverages cloud game hosting to enable players to share a complete game state (not just a specified save state) instantly to their friends or to the public. This project explores the affordances of this technology specifically for the discovery of new kinds of game design patterns and new game genres that are inherently viral and require elaborate player cooperation across massive scale audiences.
The project is still running and more contents will be added.
